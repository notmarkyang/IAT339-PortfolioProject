About This Portfolio

Ethos

This portfolio presents me as a designer-developer who works at the intersection of technical implementation and creative experience. I want to communicate that I can build systems, design for players, and contribute meaningfully to a creative team — while also being someone who cares deeply about how things look and feel.

Target Audience

Recruiters, hiring managers, and creative teams looking for junior candidates in UX design, game UX, frontend development, and interactive media. Both technical and creative audiences are addressed: the monospace type and dark theme signal technical fluency, while the project work and art speak to creative range.

Visual Identity Summary

Minimal dark interface with a single strong accent colour (Terminal Green). The design avoids decoration for its own sake — every element earns its place. The blinking cursor on the logo is the one signature detail that quietly ties the visual identity to the coder/game aesthetic without being heavy-handed.

Colour Relationships

The palette uses a near-black background family (Void, Slate, Grid Line) as its foundation, evoking a code editor or terminal environment. Terminal Green is the sole accent, chosen for its association with command-line interfaces and classic game UI, while staying readable and accessible. Frost and Dim provide two levels of text hierarchy without introducing additional hues. The palette is intentionally restrained — one accent colour keeps everything legible and avoids visual noise.

Typefaces / Fonts

Source Code Pro — Headings & UI

A monospace typeface designed by Paul D. Hunt for Adobe. Used for all headings (h1–h4), navigation, labels, tags, and code blocks. Weights used: Regular (400), SemiBold (600), Bold (700). Chosen because monospace type directly references coding environments and game engine interfaces — it communicates technical fluency without being a cliché.

Inter — Body

A sans-serif typeface designed by Rasmus Andersson, optimised for screen readability. Used for all paragraph text and longer-form content. Weight used: Regular (400). Chosen for its clean legibility at small sizes, which keeps process analysis text easy to read without competing with the monospace headings.

Signature Design Element

The blinking cursor appended to the site logo (mark_) is the portfolio's single signature detail. It is implemented in CSS using @keyframes and animation. It references both the typing cursor of a code editor and the input prompt of a game console or terminal — directly connecting the visual identity to the ethos being presented.
